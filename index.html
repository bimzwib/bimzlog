Setup Guide

How to Install BimzTools

  1. 1 Install pyRevit first — it's required before anything else. Download it from the official site:
  2. pyrevitlabs.io
  3. 2 Confirm the install — open Revit and check that a new pyRevit tab appears in the Ribbon. This confirms pyRevit is working before you continue.
  4. 3 Download and unzip BimzTools using the download button above, then extract it anywhere on your machine.
  5. 4 Run the installer — inside the extracted folder, find and run Install_BimzTools.bat
  6. Once it finishes, BimzTools will load automatically inside Revit — ready to use.
